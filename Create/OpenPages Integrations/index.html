OpenPages Integrations

Setup OpenPages with Watsonx Assistant

Prerequisite​

  • A OpenPages instance
  • A Watsonx Assistant instance
  • A Watsonx.ai project
  • A Watson Discovery instance

Integrate Watsonx Assistant with OpenPages​

In Watsonx Assistant​

  1. Navigate to Watsonx Assistant's Integrations Tab
  2. Find "Web chat", click on "Open" -> choose environment, and "Confirm"
  3. Navigate to Embed Tab and Copy the content in the text box. (This is an example)

In OpenPages​

  1. Click Administration menu > Integrations > Watson Assistant.
  2. Click Import Configuration (The button left of New +)
  3. Paste the code snippet into the Configuration string box, and then click Import.
  4. All good! You could now start interacting with your Watsonx Assistant in OpenPages.
